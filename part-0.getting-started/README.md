# Getting Started

This is a short prerequisites.

By the end of this section you should have a k6 installation.
If you already have one - you may skip it.

# Wais to check k6 installation

```sh
which k6
```

```sh
k6 --version
```

# Installing

## brew

```sh
brew install k6
```
